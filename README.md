🛠 Installation Guide
Follow these steps to get Otyiot+ up and running in seconds:

1. Prepare the Files
Download: Ensure all 6 project files are downloaded.

Folder: Place them all into a single folder on your computer (e.g., a folder named Otyiot-Extension).

2. Load into Chrome
Open Google Chrome and navigate to: chrome://extensions/

In the top right corner, toggle Developer mode to ON.

Click the Load unpacked button in the top left.

Select the folder where you saved the project files.

3. Pin for Easy Access
Click the Extensions icon (the gray puzzle piece 🧩) next to your profile picture in the Chrome toolbar.

Find Otyiot+ in the list.

Click the Pin icon (📌) next to it so it stays visible.
